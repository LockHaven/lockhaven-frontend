How to Convert to Favicon

  1. Download the SVG: Right-click on the icon above and "Save image as..." or use the SVG file at public/lock-icon.svg
  2. Convert to ICO: Use an online converter like:
    • Convertio
    • Favicon.io
    • Favicon Generator
  3. Save as favicon.ico in your public/ folder
  4. Update your layout.tsx to reference the favicon

Alternative: Use SVG as Favicon

Modern browsers support SVG favicons! You can simply add this to your src/app/layout.tsx:

<link rel="icon" type="image/svg+xml" href="/lock-icon.svg" />

Update Your Layout

Replace the existing favicon reference in your layout.tsx with:

import type { Metadata } from 'next'

export const metadata: Metadata = {
  title: 'LockHaven - Secure File Storage',
  description: 'Upload, encrypt, and store your files with military-grade security.',
  icons: {
    icon: '/lock-icon.svg',
    shortcut: '/lock-icon.svg',
    apple: '/lock-icon.svg',
  },
}
